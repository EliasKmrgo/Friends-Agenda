Friends Events API

Servicio en ejecución. Usa los endpoints REST o prueba con los enlaces rápidos.

Enlaces rápidos

  • GET /api/lugares – Lista de lugares
  • GET /api/personas – Lista de personas
  • GET /api/eventos – Lista de eventos

Ejemplos cURL

1) Crear lugar

curl -X POST http://localhost:8080/api/lugares \
  -H "Content-Type: application/json" \
  -d '{
    "direccion": "Cra 1 #23-45",
    "ciudad": "Bogotá"
  }'

2) Crear persona

curl -X POST http://localhost:8080/api/personas \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Alex",
    "fechaNac": "1990-05-20"
  }'

3) Crear evento

curl -X POST http://localhost:8080/api/eventos \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Kickoff",
    "descripcion": "Reunión inicial",
    "inicio": "2025-10-01T09:00:00",
    "fin": "2025-10-01T10:30:00",
    "lugarId": 1
  }'

4) Listar eventos

curl http://localhost:8080/api/eventos

5) Inscribir persona en evento

curl -X POST http://localhost:8080/api/eventos/1/participantes \
  -H "Content-Type: application/json" \
  -d '{
    "personaId": 1
  }'
Spring Boot sirve automáticamente este archivo desde src/main/resources/static/index.html.
