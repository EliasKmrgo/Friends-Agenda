Friends Events API

Servicio en ejecución. Usa los endpoints REST o prueba con los enlaces rápidos.

Enlaces rápidos

  • GET /api/eventos — Lista de eventos
  • GET /api/inscripciones/evento/{eventoId}

Ejemplos cURL

Crear evento

curl -X POST http://localhost:8080/api/eventos \ 
  -H "Content-Type: application/json" \ 
  -d '{
    "titulo": "Kickoff",
    "fecha": "2025-10-01",
    "duracionMin": 90,
    "direccion": "Cra 1 #23-45",
    "ciudad": "Bogotá",
    "pais": "CO"
  }'

Listar eventos

curl http://localhost:8080/api/eventos

Inscribir persona

curl -X POST http://localhost:8080/api/inscripciones \ 
  -H "Content-Type: application/json" \ 
  -d '{
    "personaId": "P1",
    "eventoId": "<ID_DEL_EVENTO>"
  }'
Spring Boot sirve automáticamente este archivo desde src/main/resources/static/index.html.
